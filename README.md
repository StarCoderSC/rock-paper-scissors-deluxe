# 🎮 Rock Paper Scissors Deluxe

An enhanced Rock Paper Scissors game built with HTML, CSS and Javascript.

Originally inspired by the SuperSimpleDev JavaScript course and expanded with additional features and improvements.

## Features

### 🎮 Gameplay

- Rock, Paper, Scissors against the computer
- Real-time result display
- Reset score button

### 📊 Statistics

- Win, losses, and ties
- Game played
- Highest wins record
- Win rate
- Current streak
- Best win streak

### 🏆 Achievements

- First Victory
- Ten Wins
- 50 Games Played
- 100 Games Played
- Win Streak Master
- Veteran
- Champion
- Unlucky
- Peacekeeper
- Rock Enjoyer

### 📈 Move Tracking

- Rock usage
- Paper usage
- Scissors usage

### Persistence

All progress is saved using Local Storage

### UI

- Dark theme
- Sidebar layout
- Hover effects
- Dynamic colors
- Locked and unlocked achievements

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

## Inspirtion

This project began as a part of the SuperSimpleDev
JavaScript Full Course and was later expanded with custom features and UI improvements.
